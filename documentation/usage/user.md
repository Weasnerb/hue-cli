# User Command Usage

[<- Back](../../README.md)

```text
  Usage: user|u [options]

  View and unregister users on bridge

  Options:

    -c, --current       Get current user
    -l, --list [name]   List registered users, using <name> as optional filter
    -d, --delete <id>   Delete registered user with <id> from bridge
    -p, --purge [name]  Delete all registered users, using <name> as optional filter
    -h, --help          output usage information

```
